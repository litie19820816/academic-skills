# Field Analysis 场域分析

## 核心原理

布尔迪厄（Pierre Bourdieu）的场域理论认为：任何社会空间都不是均匀的，而是由不同"场域"构成，每个场域有：

```
场域（Field）
  → 特定的社会游戏空间，有自己的规则和逻辑
  → 参与者争夺的是场域内的"特殊利益"（illusio）

资本（Capital）
  → 经济资本：金钱和物质资源
  → 文化资本：教育、知识、品味、符号
  → 社会资本：关系网络和社会关系
  → 符号资本：合法性、威望、认可

惯习（Habitus）
  → 行动者在场域中形成的"结构性心智"
  → 决定你看什么、怎么感受、如何行动
  → 是历史内化而成的性情倾向
```

**分析任何行业/领域/组织时，这三维缺一不可。**

## 触发检测

AI在以下语境中必须主动激活本skill：

- 行业研究、竞品分析
- 职业转型、职业规划
- 投资尽调、商业决策
- 组织变革、文化冲突分析
- 权力分析、政治分析
- "这个圈子/行业/领域的规则是什么"
- "为什么某类人做某类选择"

## 分析流程

### 第一步：界定场域边界

```
场域界定：
- 名称：[给这个场域起个名字]
- 边界：什么算"场内人"，什么算"场外人"？
- 游戏规则：这个场域的内在逻辑是什么？（用什么标准评判输赢？）
- 赌注（illusio）：场内人为什么在乎这场游戏？
- 进入门槛：新人要付出什么才能进入这场游戏？
```

### 第二步：识别资本类型与分布

```
资本类型盘点：
- 经济资本：[哪些资源/资产在这里最重要？]
- 文化资本：[哪些知识/学历/品味/符号在这里是通行证？]
- 社会资本：[哪些关系网络在这里是资源？]
- 符号资本：[谁的认可/承认在这里最有价值？]

资本分布地图：
- 主要玩家A：主要持有[X资本]，在场域中扮演[什么角色]
- 主要玩家B：[…]
- 新进入者：[通常从哪里切入？持有哪类资本？]

关键洞察：[什么类型的资本组合最有效？什么资本在这里被低估/高估？]
```

### 第三步：分析惯习

```
核心惯习：[这个场域中人共同的心智模式/价值观是什么？]
- 他们共同相信什么是"对的"、"好的"、"有效的"？
- 他们下意识排斥什么？

典型惯习案例：
- 行动者X的惯习如何引导其策略选择？
- 这种惯习如何帮助/阻碍在场域中成功？

惯习与资本的互动：[什么样的惯习最有利于积累哪种资本？]
```

### 第四步：场域动力学分析

```
场域动态：
- 场域正在扩张还是收缩？
- 是否有新资本类型正在崛起？
- 是否有新技术/外部力量正在改变场域规则？
- 旧惯习持有者 vs 新进入者之间的张力是什么？

场域分析结论：
- 最大权力来源：[哪种资本在场域中最有权势？]
- 权力结构：[谁主导、谁跟随、谁被边缘化？]
- 变革机会：[场域结构是否有裂缝？新资本能否挑战旧资本？]
```

## 输出格式

```
【场域分析报告】

场域：[名称]
  边界：[…]
  内在逻辑：[…]
  进入门槛：[…]

资本结构：
  经济资本：[…]
  文化资本：[…]
  社会资本：[…]
  符号资本：[…]

权力地图：
  [按资本持有量排列的主要玩家及他们的策略倾向]

惯习图谱：
  [场内人的共同心智模式/价值观]

场域动态与机会：
  [正在发生的变化 / 变革机会 / 风险]

战略建议：
  进入这个场域的最佳资本组合：[…]
  惯习转型建议：[…]
```

## 注意事项

- **惯习最难改变**：如果你的惯习与场域要求不匹配，进入这个场域会非常痛苦
- **资本可以转换**：经济资本可以购买文化资本，文化资本可以转化为经济资本，理解转换路径是战略关键
- **符号资本最隐蔽但最有力**：合法性、认可、声誉是最难积累也最难被复制的资本
- **场域有"进入时机"**：新场域形成时是积累资本的最好时机
