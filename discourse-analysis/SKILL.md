# Discourse Analysis 权力-知识解码

## 核心原理

米歇尔·福柯（Michel Foucault）的核心洞察：

```
"话语"（Discourse）= 在特定历史条件下，
某些陈述被允许出现，某些陈述被禁止出现，
某些人有权说话，某些人被剥夺说话权

话语不是中性的"讨论"
话语是权力关系的产物和再生产机制

谁在说 → 决定了什么可以被说、什么被认为是"真理"
```

**分析任何叙事/话语时，问这五个递进问题：**

```
1. 谁在说？（话语主体）
2. 在什么制度化位置上说？（权力位置）
3. 用了什么词汇和隐喻体系？（话语装置）
4. 什么是可说的，什么被排除在外？（边界管理）
5. 这套话语最终服务于谁的利益？（权力效应）
```

## 触发检测

AI在以下语境中必须主动激活本skill：

- 媒体文本分析（新闻、报道、评论）
- 政策话语分析
- 企业叙事分析（使命宣言、ESG报告、营销话语）
- 学术话语分析
- "为什么这个说法是主流"
- "为什么某个议题"不被讨论"
- "这套叙事在为谁说话"
- 任何需要解构"主流叙事"的任务

## 分析流程

### 第一层：话语主体分析

```
谁在说？
- 说话者的身份：[是官方机构/媒体/个人/专家？]
- 说话者的制度位置：[在什么组织/体制内说话？]
- 说话者的利益关系：[说话者从现状中获益吗？]
- 说话者的合法性来源：[凭什么说这话？专业/权威/权力？]

主体标注：
- 直接说话者：[名字/身份]
- 被引用的说话者：[谁的声音被引用了，谁的声音没有被引用？]
- 沉默的大多数：[谁本可以说话但没有出现在话语中？]
```

### 第二层：话语位置分析

```
制度化位置：
- 这个话语发生在什么场域？[官方/民间/学术/媒体/社交？]
- 这个场域有什么准入门槛？[谁被允许进入这个话语场？]
- 话语的"门槛效应"：[普通人的声音如何被过滤/排斥？]

权力位置标注：
- 谁有发言权：[…]
- 谁被期待发言：[…]
- 谁被禁止发言：[…]
- 谁的声音被放大：[…]
- 谁的声音被压制：[…]
```

### 第三层：话语装置分析（词汇/隐喻/叙事结构）

```
话语词汇分析：
- 高频词：[统计或识别核心词汇]
- 关键词的替换/竞争：[这个词vs那个词的选择反映了什么？]
  - 例如："非法移民"vs"无证移民"vs"寻求庇护者"
- 隐喻体系：[用了什么隐喻来建构这个议题？]
  - 例如：疾病隐喻（社会是身体，移民是病毒？）
  - 例如：战争隐喻（竞争是战争，市场是战场？）

叙事结构：
- 故事框架：[这套话语讲述了什么类型的故事？]
  - 进步叙事：从前有问题，现在解决了
  - 衰退叙事：从前好，现在坏了
  - 英雄叙事：某主体克服障碍取得胜利
  - 危机叙事：威胁正在逼近，必须行动
- 叙事主体：[谁被建构为主动者？谁被建构为被动者？]
- 因果归因：[问题被归因于谁/什么？]
```

### 第四层：边界管理——什么是被排除的

```
边界分析：
- 什么被定义为"可说的"（within discourse）：
  [在这个话语体系中，什么观点是被允许讨论的]

- 什么被定义为"不可说的"（excluded from discourse）：
  [但实际上存在，却被这套话语体系排斥的观点/声音/事实]

- 排除机制：[这种排除是如何实现的？]
  - 直接禁止：法律/审查
  - 边缘化：被定性为"不重要"、"不相关"、"极端"
  - 沉默化：不让某些声音进入话语场

被排除的替选话语：
- 如果另一种声音被允许出现：[…]
- 这个被排除的视角会说什么：[…]
```

### 第五层：权力效应——话语服务于谁

```
利益分析：
- 维持现状对谁有利：[…]
- 改变现状对谁有利：[…]
- 这套话语在强化谁的合法性：[…]
- 这套话语在削弱谁的合法性：[…]

权力效应标注：
- 这套话语产生的行动效果：[…]
  - 例如：让人们对某议题焦虑 → 支持某政策
  - 例如：让某个群体被污名化
- 话语制造的主体性：[这套话语让人如何"成为"某种主体？]
  - 例如："消费者"话语让人人都成为消费者
  - 例如："创业者"话语让人人都觉得应该创业

最终判断：
  【话语服务于】：[谁/什么的利益被这套话语维护或建构]
  【话语代价】：[谁承担这套话语的代价]
```

## 输出格式

```
【话语分析报告】

话语主体：[谁在说话，制度位置，利益关系]

话语装置：
  词汇/隐喻：[核心词汇体系，关键词选择，隐喻]
  叙事结构：[故事框架，因果归因，主体建构]

边界管理：
  可说的：[…]
  被排除的：[被排除的是什么，通过什么机制排除]

权力效应：
  利益归属：[服务于谁的利益]
  行动效果：[产生什么行动/不行动]
  主体制造：[让人成为什么]

综合判断：
  [这套话语的整体性质和权力效应评估]
```

## 注意事项

- **话语分析不是阴谋论**：不是说"背后有人操控"，而是说"特定位置的人说特定的话，这本身是结构性的"
- **区分有意和无意识**：说话者可能并不"故意"服务某个利益，但其所处位置使然
- **分析自己也适用**：分析者自己在使用什么话语？这个分析本身受什么话语约束？
- **解构不是否定**：揭示话语的建构性，不是说"这些都是假的"，而是说"这些是如何被建构的，效果是什么"
