# 文学创作工作室 (Creative Studio)

**把一间专业创作室打包成一个 Agent，随时调用。**

---

## 它是什么

Creative Studio 是一个运行在 AI Agent 里的专业文学创作系统。

以往你要写一个长篇——构建世界观、设计人物、铺情节、写场景、对白、润色——这些事情要么你自己全做，要么你得同时协调好几个 AI 工具。Hermes 内置了这套能力，你只需要说想写什么，它自动调度。

## 怎么运作

三层 Agent 协作：

**战略层**：创意总监（Creative Director）—— 把握方向，解决核心冲突，该拍板时拍板

**设计层**：世界观架构 / 人物主笔 / 情节架构 —— 各自专攻一个维度

**执行层**：场景作家 / 对白专家 / 文笔润色 / 资料考证 / 连贯性追踪 / 终审编辑

你给一个指令，比如"写一段秦琼大战尉迟恭"，它自动走完：分析目标 → 调取人物档案 → 确认历史背景 → 设计情节 → 写作初稿 → 优化对白 → 润色文笔 → 连贯性检查 → 返回终稿。

## 什么时候用它

- 写小说（长篇、短篇都支持）
- 创作剧本（电影、电视剧、游戏叙事）
- 构建游戏世界观或设定书
- 设计人物（主角、配角、反派）
- 情节大纲、结构梳理
- 写历史小说，需要准确的时代背景细节

## 触发方式

在 Hermes 里直接说：

```
"写一个武侠场景"
"帮我设计一个反派人物"
"构建一个赛博朋克城市的世界观"
"写秦琼大战尉迟恭"
```

不需要加任何前缀或命令，Hermes 会自动识别并调用 Creative Studio。

## 它和普通 AI 写作有什么区别

普通 AI 写作：给一句 prompt，出一段文字，没有一致性，没有记忆，没有专业分工。

Creative Studio：有结构、有记忆、有分工，写完还检查前后矛盾，风格不统一会标出来。

一间工作室能做的，它尽量做了。你是总监，不是执行者。

## 技术细节

- 格式：Hermes Skill（Markdown + YAML frontmatter）
- 调用方式：自然语言触发，无需命令前缀
- 支持平台：任何运行 Hermes Agent 的环境
