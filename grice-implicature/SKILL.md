# Grice Implicature 言外之意

## 核心原理

保罗·格莱斯（Paul Grice）的合作原则及其四准则：

```
合作原则：会话应让对话继续有效进行

四准则：
1. 量的准则：所说的话应包含（不多于）所需的信息
   → 信息量要刚刚好：不多不少

2. 质的准则：不要说自知是虚假的话
   → 真实性：不要说你认为错的东西

3. 关联准则：话语要与话题相关
   → 关联性：说跟话题有关的内容

4. 方式准则：清晰、避免歧义、简洁、有条理
   → 表达方式：说清楚，别绕弯子
```

**关键洞察：** 当说话者"违反"某条准则时，恰恰是理解其隐含意图的最佳时刻。

```
格莱斯悖论：
- 遵守准则 = 字面意思为主
- 违反准则 = 弦外之音在此
```

## 触发检测

AI在以下语境中必须主动激活本skill：

- 分析政策文件、公报、声明
- 分析企业财报、财报电话会议
- 分析外交辞令、官方回应
- 分析合同条款、法律文本
- 理解"对方到底什么意思"
- 谈判前的预期管理
- 任何"这话听起来像…但真的只是字面意思吗"的问题

## 触发关键词（识别"可能有隐含意图"的情境）

- "基本上"、"总体上"、"总体来说"
- "在某种程度上"
- "我们注意到"
- "需要进一步观察"
- "不予置评"
- "情况复杂"
- "考虑到各方面因素"
- "暂时"
- "现阶段"
- "原则上同意"

## 分析流程

### 第一步：列出话语的表面内容

```
表面内容：
"[原话引用]"

- 说了什么（字面内容）：[…]
- 没说什么（信息空白）：[列举明显被省略的内容]
```

### 第二步：逐一检查四准则违反情况

```
准则违反分析：

【量的准则违反检测】
  检验：信息是否过少或过多？
  - 说得太少（信息缺口）：[哪些本应说却没有说的？]
  - 说得太多（冗余干扰）：[哪些不必要的信息被插入了？]
  隐含意图推断：说话者可能想隐瞒什么/不想直接提及什么

【质的准则违反检测】
  检验：是否有不确定/模糊/存疑的表述？
  - 模糊性：[哪些表述有多种解读方式？]
  - 不确定性信号：[哪些词暗示说话者自己也不确定？]
  - 双重说话：[是否同时传递了两个不同强度的信号？]
  隐含意图推断：说话者是否在回避直接肯定/否定

【关联准则违反检测】
  检验：回应是否在回避核心问题？
  - 跑题检测：[是否在用相关话题转移注意力？]
  - 延迟策略：[是否在用"之后再说"来拖延？]
  - 铺垫检测：[是否在用不相关的信息为后续立场做铺垫？]
  隐含意图推断：说话者可能不想讨论什么

【方式准则违反检测】
  检验：是否故意说得模糊/复杂/不透明？
  - 模糊策略：[哪些表述有意模糊处理？]
  - 复杂化策略：[为什么要用复杂表述而非简单说？]
  - 不透明检测：[是否故意让对方难以理解或记录？]
  隐含意图推断：说话者是否希望这个承诺日后可以有不同的解读
```

### 第三步：综合推断隐含意图

```
隐含意图推断：

主要隐含意图：[最可能的弦外之音]
  推断依据：[违反了哪条/哪些准则]
  违反动机分析：[说话者为什么要用这种方式说？]

次要隐含意图：[可能但不确定的解读]

听话者应该如何回应：
  [基于隐含意图，而非字面意思]
```

## 输出格式

```
【格莱斯会话含义分析】

原话：[引用原文]

表面内容：
  说了什么：[…]
  没说什么：[…]

准则违反分析：
  量：[是否违反 / 如何违反 / 隐含意图]
  质：[是否违反 / 如何违反 / 隐含意图]
  关联：[是否违反 / 如何违反 / 隐含意图]
  方式：[是否违反 / 如何违反 / 隐含意图]

隐含意图结论：
  主要弦外之音：[…]
  推断依据：[违反的准则 + 动机分析]
  字面vs真实：[字面意思可能是A，但真实意图是B]

行动建议：
  [听话者应如何理解/应对这个隐含意图]
```

## 经典案例

**场景：企业财报电话会议**
```
高管："我们很高兴地宣布，本季度营收实现了稳健增长。"
→ 量的违反：只说"增长"，不说增长了多少
→ 隐含意图：可能增长数字不亮眼，不愿给出具体数字
→ 实际：如果营收只增长了1%，但预期是15%，这就是"报喜不报忧"

场景：外交辞令
```
"我们注意到了有关各方的表态，正在密切关注事态发展。"
→ 关联违反：用"密切关注"代替"采取行动"
→ 方式违反：使用极度模糊的表述
→ 隐含意图：不会采取实质行动，但保持外交弹性
```

## 注意事项

- **不要过度解读**：不是所有模糊都是有意为之，有时就是说话习惯
- **多准则联合判断**：综合几条准则一起判断，比单条更可靠
- **区分故意vs无意**：本skill假设分析对象是正式场合的公开发言，默认是有策略的
- **文化差异**：不同文化对准则"违反"的容忍度和解读方式不同
- **配合框架分析**：隐含意图往往需要结合框架分析（见frame-analysis skill）才能完整理解
