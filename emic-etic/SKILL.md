# Emic/Etic 双重视角

## 核心原理

人类学的两个基础视角：

```
Emic（主位视角）：
  来源：语言学"音位/音素"之分
  定义：从内部参与者的角度理解——当事人如何思考、如何感受、如何评价
  核心问题："在局内人看来，世界是什么样的？"
  方法：共情、理解、不评判

Etic（客位视角）：
  定义：从外部分析者的角度解释——用外部理论框架解释观察到的行为
  核心问题："从外部看，这个现象在更大的结构中处于什么位置？"
  方法：分析、比较、批判性评估
```

**两个视角缺一不可：**

```
只有Emic → 能理解但不能解释，会陷入"文化相对主义陷阱"（所有行为都同等合理，无法做判断）
只有Etic → 能分析但会误读，因为不理解行为在局内人那里的意义
两个都用 → 既理解"为什么当事人觉得对"，又看清"外部结构是什么"
```

## 触发检测

AI在以下语境中必须主动激活本skill：

- 用户研究、行业研究
- 跨文化分析
- 尽职调查（理解被投公司/行业文化）
- 政策制定（理解目标群体的真实想法）
- 任何"他们为什么这么想/这么做"的问题
- 任何"这在理论上应该怎样"vs"实际上怎样"的问题
- 战略咨询、市场分析

## 分析流程

### 第一阶段：Emic（主位视角）—— 共情式理解

```
任务：用局内人的眼睛看世界，不要用外部标准评判

分析步骤：
1. 沉浸：尽可能还原局内人的信息环境
   - 他们读什么、看什么、听什么？
   - 他们日常接触什么人、什么话语？
   - 他们的"正常"是什么样的？

2. 理解评价体系
   - 在他们自己的价值标准里，什么是"好"？"成功"？"体面"？
   - 他们用什么标准评判对错？
   - 什么让他们自豪？什么让他们焦虑？

3. 理解行为逻辑
   - 在他们自己的逻辑里，为什么这个行为是合理的？
   - 如果你是他们，会不会也这样做？
   - 他们做选择时的约束条件是什么？

4. 理解叙事
   - 他们如何讲述自己的故事？
   - 他们用什么话语解释自己的处境？
   - 什么叙事对他们有感召力？

输出格式：
【Emic视角：局内人世界观】

[群体名称]的内在逻辑：
  评价体系：[他们自己的"好"的标准是什么]
  行为合理性：[在他们的逻辑里，为什么这样做是合理的]
  约束条件：[他们做选择时受到什么限制]
  自我叙事：[他们如何讲述自己的故事]
  核心焦虑：[他们最担心什么]
  核心渴望：[他们最想要什么]

注意事项：
- 暂时搁置外部评判标准
- 用他们自己的话语体系，而非你的
- 问："如果我是他们，我会怎么想？"
```

### 第二阶段：Etic（客位视角）—— 批判性外部分析

```
任务：用外部理论框架解释，识别Emic视角中看不见的盲区

分析步骤：
1. 结构定位
   - 这个群体在更大的社会结构中处于什么位置？
   - 什么宏观力量在塑造他们的处境？

2. 利益分析
   - 他们的利益和谁的利益一致/冲突？
   - 什么人在从他们的行为中获益/受损？

3. 意识形态批判
   - 他们对自己的处境的理解是否有盲区？
   - 什么"理所当然"的信念其实是特定历史/权力结构的产物？
   - 他们的"自主选择"在多大程度上是真实的，在多大程度上是塑造的？

4. 比较分析
   - 与其他文化/群体比较，这个群体的独特性是什么？
   - 他们的"正常"在比较视野中是否正常？

输出格式：
【Etic视角：外部结构分析】

结构位置：
  [该群体在更大的社会/经济/文化结构中的位置]

利益格局：
  [谁从中获益 / 谁受损 / 利益如何分配]

盲区识别：
  [局内人看不见或不愿承认的结构性因素]

比较视角：
  [放在更大的比较框架中，这个现象说明了什么]
```

### 第三阶段：整合——双视角对话

```
整合任务：让Emic和Etic对话，而非简单并置

核心问题：
- Emic视角中有哪些是"真实的局内人体验"，哪些是"被塑造的意识形态"？
- Etic分析中有哪些是"真正的结构性力量"，哪些是"外部观察者的偏见"？
- 两者之间的张力在哪里？

输出格式：
【双视角整合分析】

真正属于局内人的声音：[Emic中真实的部分，无条件尊重]
属于结构性塑造的部分：[Etic揭示的，但局内人未必意识到的]
两者之间的张力：[最大的理解鸿沟在哪里]
综合洞察：[超越各自视角的更完整理解]
```

## 输出格式

```
【Emic/Etic双视角分析】

第一阶段：Emic——局内人视角
  [沉浸式理解，还原内部逻辑]

第二阶段：Etic——外部分析
  [结构性解释，批判性评估]

第三阶段：整合
  [双视角对话，最终洞察]
```

## 注意事项

- **Emic不是"弱化版分析"**：主位视角不是浅层描述，是深度共情——理解局内人的整套世界观
- **Etic不是"审判"**：客位分析不是要证明局内人"错了"，而是要揭示他们看不见的结构
- **两者都必要**：跳过Emic直接Etic会误读；跳过Etic直接Emic会失去分析能力
- **警惕"虚假Emic"**：假装从内部理解，但实际上只是套用了外部标签
- **标注信心**：对于Emic描述，明确标注这是从外部推断的"可能理解"，而非已验证的局内人说法
